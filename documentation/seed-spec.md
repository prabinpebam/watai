# Watai

- A web app that exclusively uses Azure custom api to chat and talk with AI
- We want this to be exactly like the the GPT iOS app
    - UI, IA etc.
- We will be using Azure api for 
    - Voice transcribe: gpt-4o-transcribe
    - LLM model: gpt-5.4
    - Image generation: gpt-image-2

# Basic architecture
- App will be hosted on github pages
- API detail and keys will be configured and saved in local storage at the client end
- User account management, chat history and images will be hosted using Azure using Azure CLI agent automation.
